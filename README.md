# Sports Hall Booking System

## Project Overview
The Sports Hall Booking System is developed using ServiceNow Service Portal to automate sports facility reservations.

## Features
- Sports Hall Management
- Booking Management
- Booking Validation
- Approval Workflow
- Email Notifications
- Payment Calculation
- Refund Calculation
- Service Portal Pages
- Service Portal Widgets

## Technologies Used
- ServiceNow
- Flow Designer
- Business Rules
- Service Portal
- Email Notifications
