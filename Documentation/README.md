# Sports Hall Booking System

## Project Overview
The Sports Hall Booking System is developed using ServiceNow Service Portal to automate the sports hall reservation process. It enables users to book sports facilities online, validates bookings to prevent conflicts, manages approvals, calculates payments and refunds, and sends email notifications.

## Features
- Sports Hall Management
- Sports Hall Booking Management
- Booking Validation & Availability Check
- Approval Workflow
- Email Notifications
- Automatic Payment Calculation
- Refund Calculation (10% Cancellation Deduction)
- Service Portal Pages
- Service Portal Widgets
- Business Rules
- Flow Designer Automation
